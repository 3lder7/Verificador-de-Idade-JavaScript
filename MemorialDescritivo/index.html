Verificador de Idade

Análise de Requisitos e Casos de Uso

Instruções: Este diagrama mostra a estrutura UML completa do aplicativo Verificador de Idade, incluindo casos de uso, requisitos funcionais e não funcionais.

Necessidades do Usuário

  • Facilidade de uso: Interface intuitiva e simples
  • Rapidez: Resultado imediato após inserção dos dados
  • Precisão: Cálculo correto da idade e classificação etária
  • Acessibilidade: Funcionamento em dispositivos móveis
  • Feedback visual: Apresentação clara do resultado
  • Privacidade: Dados processados localmente sem envio para servidor

Objetivos do Sistema

  • Calcular idade com base no ano de nascimento
  • Classificar usuário em faixa etária específica
  • Apresentar resultado de forma visual com imagem
  • Validar dados de entrada para evitar erros
  • Oferecer experiência responsiva em diferentes dispositivos
  • Manter histórico visual da última consulta

Requisitos do Sistema

Requisitos Funcionais

  1. RF01: O sistema deve permitir inserção do ano de nascimento
  2. RF02: O sistema deve permitir seleção do gênero (masculino/feminino)
  3. RF03: O sistema deve validar se o ano é válido (não futuro)
  4. RF04: O sistema deve calcular a idade baseada no ano atual
  5. RF05: O sistema deve classificar a faixa etária:
    • Criança (1-11 anos)
    • Adolescente (12-18 anos)
    • Jovem (19-29 anos)
    • Adulto (30-49 anos)
    • Idoso (50+ anos)
  6. RF06: O sistema deve exibir imagem correspondente à faixa etária e gênero
  7. RF07: O sistema deve mostrar mensagem de erro para dados inválidos

Requisitos Não Funcionais

  1. RNF01: Usabilidade - Interface intuitiva com tempo de aprendizado < 2 minutos
  2. RNF02: Responsividade - Funcionamento em dispositivos móveis e desktop
  3. RNF03: Performance - Tempo de resposta < 1 segundo
  4. RNF04: Confiabilidade - Disponibilidade de 99%
  5. RNF05: Segurança - Processamento local sem envio de dados
  6. RNF06: Compatibilidade - Funcionamento nos principais navegadores
  7. RNF07: Acessibilidade - Contraste adequado e navegação por teclado
